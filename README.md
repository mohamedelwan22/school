# روضة الطالبين  

رَوضُةُ الطَّالِبِينَ
مدرستنا الدينية هي منارة علم وإيمان، تهدف إلى تربية جيل واعٍ بدينه، معتزّ بهويته، ومتمسك بقيمه في زمن التحديات الصعبة.
نُقدّم تعليمًا شرعيًا متينًا مبنيًا على القرآن الكريم والسنة النبوية، بأسلوب تربوي عصري يجمع بين الأصالة والمعاصرة، على يد نخبة من العلماء والمعلمين المتخصصين.
